# Deleter

---

__Описание:__ 

Небольшое десктопное приложение для внутренних нужд компании.
Программа написана на Java, JavaFX. Программа запускается из exe файла, работает на всех версиях Windows.

__Назначение:__ 

Программа предназначена для сортировки управляющих программ в серверных папках станка по срокам давности, а именно для быстрой сортировки и удаления устаревших файлов.

__Функции:__ 


* Выбор каталога 
	* при наличие доступа к DNC серверу стартовое окно выбора - корень DNC
	* в противном случае стартовый каталог системы
* Выбор временного интервала сортировки. Доступны: 
	* 3 дня 
	* 1 неделя
	* 2 недели
	* 3 недели
	* 1 месяц
	* 3 месяца
* Поиск файлов по имени
	* при наличие доступа к DNC серверу поиск осуществляется по всем папкам сервера для наглядного отображения в каких станках уже имеется такая программа
	* Поиск может происходить по частичному совпадению (от 3 символов)
	* Поиск осуществляется по кешу имен файлов в памяти программы, из за чего скорость поиска очень высокая.
* Отсортированные файлы возможно удалить нажатием одной кнопки или переместить в новую (выбранную) папку.
* Состояние кэша и доступность DNC:
	* Стправа от лейбла "DNC" отображается значек состояния. Возможны 3 варианта: "В процессе кеширования", "Завершено" и "Недоступно" 
	* При нажатии на лейбл "DNC" происходит принудительное обновление кэша
	* При кешировании программа перебирает папки сервера и запоминает список файлов в них, не производя никакой обработки, из за чего кеширование происходит практически мгновенно для пользователя.
	* Кеширование проходит в отдельном потоке, не оказывыя влияния на основной поток программы.
	* При недоступности сервера запускается отдельный поток, регулярно проверяющий доступность сервера и запускающий кеширование при обнаружении доступа.
* Таблицы:
	* Таблицы файлов имеют 3 колонки в обычной работе и 4 колонки в режиме поиска (добавляется колонка расположения файла). 
	* Таблицы полностью интерактивны, имеют сортировку по любому столбцу.
	* Взаимодействие со строками таблиц при помощи мыши:
		* 1 клик - выделение
		* клик по кнопкам <- или -> для перемещения файлов между таблицами (ручная сортировка)
		* 2 клик - открытие файла программой, установленной по умолчанию для этого типа файлов в системе.
	* Взаимодействие со строками таблиц при помощи клавиатуры:
	 	* стрелки "вверх", "вниз" - перемещение курсора
		* стрелки "влево", "вправо" - перемещения файлов между таблицами (ручная сортировка)
		* enter - открытие файла программой, установленной по умолчанию для этого типа файлов в системе.
* Дополнительно:
	* Вверху (хлебные крошки): отображение текущего каталога.
	* Над таблицами: подсчет колличества файлов после сортировки (слева, справа и всего)
	* Датой файла считается момент помещения файла в выбранную папку, дополнительно отображается колличество дней с момента копирования.
